Features

Ship code
with agents

A complete workspace for orchestrating AI agents — from planning to pull request.

Parallel agents with isolated worktrees

Run dozens of agents simultaneously. Each gets its own git branch and worktree — complete isolation, zero conflicts.

Plan before you build

Claude reads your code and proposes a step-by-step plan. Nothing executes until you approve.

1. Add export button to toolbar
2. Create CSV serializer
3. Wire up click handler
4. Add tests

AI code review

PR reviews with inline comments, severity levels, and one-click submit to GitHub.

Interactive questions

Claude asks you questions mid-task. Answer inline in the chat — no terminal switching.

Message queue

Stack follow-ups while agents work. They send automatically when each agent finishes.

Full workspace per agent

Chat, file diffs, integrated terminal. Track changes and review code from one place — on web, desktop, and mobile.

Team agents

Claude can spawn sub-agents that work in parallel on different parts of a task, then combine results. Automatic orchestration.
