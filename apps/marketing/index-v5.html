Features

Everything you need to ship code with agents

From planning to pull request, in one workspace.

Parallel agents

Run dozens of Claude Code agents simultaneously, each in its own isolated git worktree.

Plan mode

Review the plan before execution. Approve, dismiss, or iterate — nothing runs until you say go.

AI code review

Automated PR reviews with inline comments, severity levels, and one-click submit to GitHub.

Interactive questions

Claude can ask you questions mid-task. Answer inline in the chat, no context switching.

Message queue

Queue follow-up messages across agents. They send automatically when each agent finishes.

Team agents

Claude spawns sub-agents that work in parallel on parts of a task, then combine results.
