Features

Everything you need to
ship code with agents

A complete workspace for managing AI coding agents — from planning to pull request.

Parallel agents

Spin up dozens of Claude Code agents at once. Each runs in its own git worktree with an isolated branch — no conflicts, no waiting.

Add CSV export
Fix login redirect
Refactor auth
Update API docs
Add dark mode
DB migration

Plan mode

Review the plan before execution. Approve, dismiss, or ask for changes — Claude won't write a line until you say go.

AI code review

Automated PR reviews with inline comments, severity levels, and one-click submission to GitHub.

Interactive questions

Claude can ask you questions mid-task. Answer inline in the chat — no context switching, no terminal juggling.

Queue messages across agents

Stack up follow-ups while agents are working. Messages queue per-agent and send automatically when each one finishes — across any number of parallel agents.

How it works

Three commands to ship

Install the CLI, spin up agents, and let them work while you review.

1

Install

One command to get the server running. Works on Mac, Linux, and Windows.

npx huxflux start
2

Create agents

Describe the task. Each agent gets its own branch and worktree automatically.

"Add CSV export to the devices table"
3

Review & ship

Review diffs, approve plans, submit PRs — all from the same UI. Ship when ready.

✓ PR #142 merged
