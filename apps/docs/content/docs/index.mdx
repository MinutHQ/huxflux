Huxflux lets you run and manage multiple Claude coding agents from any browser or device. Agents work on isolated git branches, and you can monitor chat history, file diffs, and terminal output in one view.

<DownloadButtons />

## Quick start

```sh
curl -fsSL https://raw.githubusercontent.com/MinutHQ/huxflux/main/install.sh | bash
```

One command installs the server, runs setup, and optionally installs the desktop app. Or see the [full installation guide](/docs/getting-started/installation).

## Features

- **Any browser, any device** — access your agents from a browser, desktop app, or mobile
- **Multiple agents at once** — run many Claude agents in parallel, each on its own branch
- **Chat, diffs & terminal** — full conversation history, live file diffs, and terminal output
- **Self-hosted** — runs on your own machine; your code and keys never leave your infrastructure
